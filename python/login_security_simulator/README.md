# Python Login Security Simulator Jan 4, 2026

## Description
This project is a beginner Python program that simulates a secure login system with password validation, attempt limits, and basic logging. It was created as a learning exercise for cloud security and authentication logic.

---

## Features
- User-defined password
- Configurable maximum login attempts
- Lockout after too many failed attempts
- Logs failed attempts to a file with timestamp and attempted password
- Security-focused logic for practice in cloud security

---

## How to Use
1. Clone or download this repository
2. Open login_system.py in Python (VS Code or any IDE)
3. Run the script
4. Enter a password and the maximum number of allowed attempts
5. Follow the on-screen prompts
